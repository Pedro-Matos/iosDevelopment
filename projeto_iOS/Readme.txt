REQUIREMENTS
------------

necessário ter o xcode atualizado
Em qualquer diretório do sistema operativo:
▶ sudo gem install cocoapods
▶ sudo gem install cocoapods-deintegrate
▶ sudo gem install cocoapods-clean

passar para dentro do directório GeoMapping-Swift
▶ pod install   (pode demorar por volta de 5/7 minutos da primeira vez) (vai criar um novo workspace)
projecto do xcode que deve ser corrido: GeoMapping-Swift.xcworkspace



Caso apos ver o projeto compilado, quiser remover os ficheiros compilados para guardar o projecto com menos tamanho:

dentro do directorio GeoMapping-Swift
▶ pod deintegrate
▶ pod clean
